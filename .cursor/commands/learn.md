# Learning speichern

Etwas Wichtiges wurde gelernt.

1. Frage: "Was hast du gelernt?" (falls nicht angegeben)
2. Frage: "In welchem Kontext?" (Projekt, Technologie)
3. Speichere mit `remember`:
   - type: "learning"
   - importance: basierend auf Nützlichkeit
   - tags: relevante Technologien/Projekte
4. Format:
   ```
   LEARNING: [Was funktioniert/nicht]
   KONTEXT: [Wann relevant]
   LÖSUNG: [Falls vorhanden]
   ```
5. Bestätige
