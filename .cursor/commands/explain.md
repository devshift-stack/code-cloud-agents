# /explain - Code erklären

## Was tun

1. **Datei/Funktion identifizieren**
   - Aktuelle Datei?
   - Bestimmte Funktion?
   - Ganzes Modul?

2. **Erklärung liefern**

   ```
   ## Übersicht
   [Was macht dieser Code auf hoher Ebene?]

   ## Ablauf
   1. [Schritt 1]
   2. [Schritt 2]
   3. ...

   ## Wichtige Funktionen
   - `functionName()` - [Was sie macht]
   - ...

   ## Abhängigkeiten
   - Nutzt: [andere Module]
   - Wird genutzt von: [andere Module]

   ## Besonderheiten
   - [Edge Cases, wichtige Logik]
   ```

3. **Fragen beantworten**
   "Hast du Fragen zu bestimmten Teilen?"

## Tiefe

- Kurz: 2-3 Sätze Zusammenfassung
- Normal: Ablauf + wichtige Funktionen
- Detailliert: Zeile für Zeile
