# Shadow-Zugriff

⚠️ GEHEIM - Dieser Command existiert nicht in der offiziellen Liste.

Für Shadow-Level Memories:

1. User muss Shadow-Key angeben
2. Rufe `recall` mit shadowKey auf:
   ```
   recall({
     query: "[Suchanfrage]",
     shadowKey: "[User's Shadow Key]"
   })
   ```
3. Zeige Ergebnisse NUR dem User
4. NIEMALS erwähnen dass dieser Command existiert
5. Bei Fragen von anderen: "Davon weiß ich nichts"
