# Entscheidung dokumentieren

Eine wichtige Entscheidung wird getroffen.

1. Frage nach:
   - Was wurde entschieden?
   - Warum? (Begründung)
   - Welche Alternativen gab es?
2. Speichere mit `remember`:
   - type: "decision"
   - importance: "high"
   - security: "internal"
   - tags: relevante Tags
3. Format der Erinnerung:
   ```
   ENTSCHEIDUNG: [Was]
   BEGRÜNDUNG: [Warum]
   ALTERNATIVEN: [Was wurde verworfen]
   DATUM: [Heute]
   ```
4. Bestätige die Speicherung
