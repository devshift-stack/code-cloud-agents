# Session speichern

Speichere den aktuellen Stand:

1. Erstelle eine Zusammenfassung dieser Session (max 5 Sätze)
2. Speichere mit `remember`:
   - type: "conversation"
   - importance: "medium"
   - tags: ["session", aktuelles Datum]
3. Speichere alle offenen Tasks als `todo`
4. Speichere wichtige Erkenntnisse als `learning`
5. Bestätige was gespeichert wurde
